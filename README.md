# Brevitas Homebrew Tap

Homebrew formulae for [Brevitas](https://github.com/Brevitas-ai/brevitas) —
middleware that routes your AI coding assistants through a local optimization
proxy.

## Install

```sh
brew install Brevitas-ai/brevitas/brevitas
# or
brew tap Brevitas-ai/brevitas
brew install brevitas
```

Then:

```sh
brevitas install
```

## Bleeding edge

```sh
brew install --HEAD Brevitas-ai/brevitas/brevitas
```
